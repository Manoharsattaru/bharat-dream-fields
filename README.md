# Bharat Dream Fields — Premium Website

A complete premium website redesign for **Bharat Dream Fields Private Limited** — an agriculture consultancy, training, and export company based in Srikakulam, Andhra Pradesh, India.

## ✨ Features

- 🌙 **Dark luxury design** — Deep forest green + harvest gold palette
- 🖼️ Full-screen hero with AI-generated farm background & floating particles
- 📊 Animated stat counters on scroll
- 🃏 6 glassmorphism service cards
- 💰 Transparent pricing cards with Razorpay payment links
- 📱 Fully responsive — mobile hamburger menu included
- ⚡ Smooth scroll-reveal animations (IntersectionObserver)
- 🎯 Contact form with client-side validation
- 🔗 Social links: Instagram, LinkedIn, YouTube
